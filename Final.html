How to copy into Outlook

Outlook for Windows

  1. Update the employee fields, then click Apply.
  2. Click Copy Signature with Links. If needed, use Select Manually and press Ctrl+C.
  3. Open Outlook: File > Options > Mail > Signatures.
  4. Create or select a signature, paste with Ctrl+V, then save.

Outlook for Mac

  1. Update the employee fields, then click Apply.
  2. Click Copy Signature with Links. If needed, use Select Manually and press ⌘+C.
  3. Open Outlook: Outlook > Settings > Signatures.
  4. Create or select a signature, paste with ⌘+V, then set it as default if required.
